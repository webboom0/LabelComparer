AsyncAlert 데모

AsyncAlert 시스템 데모

기본 알림 (Basic Alerts)

간단한 알림 메시지를 표시합니다.

// 성공 알림 await asyncAlert.success('성공!', '작업이 완료되었습니다.'); // 에러 알림 await asyncAlert.error('오류!', '문제가 발생했습니다.'); // 경고 알림 await asyncAlert.warning('경고!', '주의가 필요합니다.'); // 정보 알림 await asyncAlert.info('정보', '참고사항입니다.');

확인 다이얼로그 (Confirm Dialog)

사용자의 확인을 받는 다이얼로그입니다.

// 기본 확인 const result = await asyncAlert.confirm('확인', '정말 진행하시겠습니까?'); if (result.isConfirmed) { console.log('사용자가 확인을 클릭했습니다.'); } else { console.log('사용자가 취소를 클릭했습니다.'); } // 커스텀 확인 const result = await asyncAlert.fire({ title: '삭제 확인', text: '이 항목을 삭제하시겠습니까?', icon: 'warning', confirmButtonText: '삭제', confirmButtonColor: 'danger', showCancelButton: true, cancelButtonText: '취소' });

입력 다이얼로그 (Input Dialog)

사용자로부터 입력을 받는 다이얼로그입니다.

// 텍스트 입력 const result = await asyncAlert.prompt('이름 입력', '사용자 이름을 입력하세요:', '홍길동'); if (result.isConfirmed) { console.log('입력된 이름:', result.value); } // 커스텀 입력 const result = await asyncAlert.fire({ title: '이메일 입력', text: '이메일 주소를 입력하세요', input: 'email', inputPlaceholder: 'example@email.com', showCancelButton: true });

커스텀 다이얼로그 (Custom Dialog)

고급 옵션을 사용한 커스텀 다이얼로그입니다.

// 커스텀 다이얼로그 const result = await asyncAlert.fire({ title: '커스텀 제목', text: '커스텀 메시지', icon: 'question', showCancelButton: true, confirmButtonText: '저장', cancelButtonText: '취소', confirmButtonColor: 'success', cancelButtonColor: 'secondary', width: '500px' }); // 로딩 다이얼로그 const result = await asyncAlert.fire({ title: '처리 중...', text: '잠시만 기다려주세요', showLoaderOnConfirm: true, allowOutsideClick: false, allowEscapeKey: false });
